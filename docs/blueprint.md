# **App Name**: AcademIQ

## Core Features:

- Student Authentication: Enable secure student login using Firebase Authentication.
- Admin Authentication: Enable secure admin login using Firebase Authentication.
- Real-time Notice Board: Display real-time notices using Firestore.
- Academic Calendar: Show important dates like holidays, deadlines, and events, stored in Firestore.
- Exam Schedule: Display the exam schedule with dates, times and locations stored in Firestore.
- Mobile Responsiveness: Ensure the app is fully responsive and works well on different devices and screen sizes.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke trust and knowledge.
- Background color: Light gray (#F0F8FF), very desaturated blue to ensure readability and calm the learning environment.
- Accent color: Electric blue (#7DF9FF), to create emphasis and differentiate elements on the page.
- Body and headline font: 'PT Sans', sans-serif, combines modernity with approachability.
- Use flat, line-based icons in a consistent style.
- Use a clean and modern layout with clear sections for each type of information.
- Subtle animations when new notices or events appear.
